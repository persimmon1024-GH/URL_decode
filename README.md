# URL Decoder / Encoder v2.0

多言語文字（UTF-8など）を含むURLのエンコード・デコードを簡単に行える、単体HTML構成のツールです。

## 概要

WebサイトのURLに含まれる `%XX` 形式のエンコード済み文字列を元の読みやすい形式にデコードしたり、逆に日本語などの文字列をURL安全な形式にエンコードしたりすることができます。

## 主な機能

- **3つの変換モード**:
  - **Decode**: `%XX` 形式の文字列を元の文字に復元します。
  - **Encode**: 文字列を完全にURLエンコードします。
  - **Smart Encode**: フルURLに対して、プロトコルやドメイン部分は維持したまま、パスやクエリパラメータ内の多言語文字のみを賢くエンコードします。
- **モダンなUI/UX**:
  - **テーマ切り替え**: ライトモードとダークモードに対応しています。
  - **レスポンシブデザイン**: PCだけでなくスマートフォンでも快適に操作できます。
  - **履歴機能**: 直近5件の変換履歴をブラウザに保存し、いつでも復元できます。
  - **スマート提案**: 入力内容を分析し、最適な変換モードを提案します。
- **実用的なツール**:
  - **クリップボード連携**: ペーストボタンやコピーボタンを完備しています。
  - **印刷・PDF対応**: 変換結果を綺麗に印刷またはPDF保存するための専用スタイルを適用しています。

## 特徴

- **完全オフライン動作**: 外部ライブラリやCDN、外部通信を一切使用していません。セキュリティが重要な環境でも安心して使用できます。
- **インストール不要**: `url_encoder_decoder_v2.html` ファイルをブラウザで開くだけですぐに使用可能です。
- **軽量**: 全てのロジック、スタイル、UIが一つのHTMLファイルに凝縮されています。

## 使用方法

1. `url_encoder_decoder_v2.html` を任意のWebブラウザ（Chrome, Edge, Firefox, Safariなど）で開きます。
2. 「Input」エリアに変換したい文字列を入力または貼り付けます。
3. 変換モードを選択し、「Transform」ボタンをクリックするか、`Ctrl / Cmd + Enter` を押します。
4. 「Output」エリアに変換結果が表示されます。

## 技術仕様

- **言語**: HTML5, Vanilla CSS, Vanilla JavaScript (ES6+)
- **ストレージ**: ブラウザの `localStorage`（履歴およびテーマ設定の保持に使用）
- **依存性**: なし（外部通信なし）
