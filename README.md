# Nexus URL - Professional Encoder / Decoder

多言語文字（UTF-8など）を含むURLのエンコード・デコードを、プレミアムな操作体験で提供する次世代の変換ツールです。

## 概要

`Nexus URL` は、WebサイトのURLに含まれる `%XX` 形式のエンコード済み文字列をデコードしたり、逆に日本語などの文字列をURL安全な形式にエンコードしたりするための、洗練された単体HTMLアプリケーションです。

## 主な機能

- **3つの変換モード**:
  - **Decode**: `%XX` 形式の文字列を元の文字に復元。
  - **Encode**: 文字列を完全にURLエンコード。
  - **Smart Encode**: フルURLに対して、プロトコルやドメインを維持したままパスやクエリのみを賢くエンコード。
- **プレミアム UI/UX**:
  - **グラスモーフィズム**: 磨りガラスのような透明感と depth（深み）のあるモダンなデザイン。
  - **SVG アイコン**: すべてのボタンに直感的なアイコン（Swap, Copy, Paste 等）を配置。
  - **スマート提案バナー**: 入力内容を自動分析し、最適な変換モードを提案。
  - **洗練されたタイポグラフィ**: `Outfit` と `JetBrains Mono` による高い視認性。
- **インテリジェントなテーマ管理**:
  - **OS設定同期**: OSのダークモード/ライトモード設定を自動検知し、リアルタイムで同期。
  - **永続化**: 最後に選択したテーマをブラウザに記憶。
- **実用的な操作系**:
  - **高度な履歴管理**: 直近8件の履歴をカード形式で保持し、ワンクリックで復元。
  - **クリップボード連携**: ペースト・コピーボタン完備。
  - **PDF対応**: 変換結果を美しく出力するためのプリントスタイル。

## 特徴

- **完全オフライン動作**: 外部ライブラリ、CDN、外部通信を一切排除。究極のプライバシーとセキュリティ。
- **ポータブル**: `url_encoder_decoder_v2.html` ファイル1つでどこでも動作。
- **ゼロ・ランタイム依存**: ブラウザがあれば、設定もサーバーも不要です。

## 使用方法

1. `url_encoder_decoder_v2.html` をブラウザで開きます。
2. 「Input」エリアに文字列を貼り付けます。
3. `Ctrl / Cmd + Enter` または中央の変換ボタン（→）で変換。
4. 「Copy Result」で結果を取得、または「Open」ボタンでURLを直接開きます。

## 技術仕様

- **Language**: HTML5, Vanilla CSS, Modern JavaScript
- **Fonts**: Inter, Outfit, JetBrains Mono (via Google Fonts)
- **External Dep**: None (Single-file self-contained)
